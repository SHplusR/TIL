2. ORDER BY '기준 컬럼' -> 어느 순으로 진행할것인가를 알려줌.
 DESC -> 내림차순 ex) 10,7,5,3,1
 asc -> 오름차순 ex)1,3,5,7,10
3. order by 뒤에 생략 시 기본은 asc
4. order by로 다중정렬시 순서대로 진행된다.SHIPMENT_ID는 TOTAL_ORDER가 같은 같에 한하여 오름차순으로 정렬된다.
